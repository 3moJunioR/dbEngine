# 🗄️ dbEngine

A **mini database engine** implemented in **C++**, built to practice core concepts of **data structures**, **file handling**, and **basic database operations** without relying on external database systems.

This project simulates how a simple database engine works internally, including storing records, searching, and managing data efficiently.

---

## 🚀 Features

- 📁 File-based data storage
- ➕ Insert records
- 🔍 Search records
- ❌ Delete records
- 📄 Display stored data
- 🧠 Uses core Data Structures concepts
- ⚡ Lightweight and fast (no external dependencies)

---

## 🛠️ Technologies Used

- **Language:** C++
- **Concepts:**
  - Data Structures
  - File Handling
  - Object-Oriented Programming (OOP)
  - Pointers & Memory Management

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/3moJunioR/dbEngine.git
```

2. Navigate to the project folder:

```bash
cd dbEngine
```

3. Compile the project:

```bash
g++ main.cpp -o dbEngine
```

4. Run the executable:

```bash
./dbEngine
```
